# FEU UAAP Season 88 | Elite Roster Experience

[![Next.js 16](https://img.shields.io/badge/Next.js-16-black?style=for-the-badge&logo=next.js)](https://nextjs.org/)
[![Tailwind CSS 4](https://img.shields.io/badge/Tailwind_CSS-4.0-38B2AC?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com/)
[![Framer Motion](https://img.shields.io/badge/Framer_Motion-12.0-ff69b4?style=for-the-badge&logo=framer)](https://www.framer.com/motion/)

A high-performance, responsive athlete showcase built for the **FEU Tamaraws** (UAAP Season 88). This project prioritizes immersive UX, 3D interactivity, and accessibility (WCAG 2.1) to create a premium "Elite Roster Database" prototype.

---

## 🌟 Key Features

### 🏆 3D Pro-Scout Carousel
An immersive, hardware-accelerated 3D carousel that leverages **Framer Motion** and **CSS Perspective**. 
- **Dynamic Focus**: Automatic centering and scaling based on scroll position.
- **Micro-Interactions**: Hover-to-reveal action shots for every athlete.
- **Hardware Sync**: Smooth 60FPS performance on mobile and desktop.

### 🌪️ Kinetic Typography & Branding
A design system inspired by elite athletic aesthetics:
- **Infinite Marquee**: A custom, performant ticker using CSS keyframes.
- **Dynamic Backgrounds**: Layered mesh glows, tech-grids, and scroll-responsive ghost watermarks.
- **Brand Consistency**: Precise implementation of FEU's Green & Gold palette with modern gradients.

### ♿ Accessibility First (ISO 9241 compliant)
Designed with inclusivity as a core requirement:
- **Keyboard Navigation**: Full support for arrow keys and focus indicators.
- **Screen Reader Optimized**: Semantic HTML and descriptive ARIA labels.
- **Reduced Motion**: Respects OS-level motion preferences.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Animations**: [Framer Motion 12](https://www.framer.com/motion/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Architecture**: Atomic component structure with centralized data management (`players.json`).

---

## 📐 Responsive Engineering

The website is engineered with a **Mobile-First** approach:
- **Touch-Optimized**: Native-feel scrolling on mobile with snap-points.
- **Adaptive Layouts**: Seamless transition from a sleek mobile view to an expansive 3D desktop experience.

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/binsacedillo/feu-tamaraws-roster.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Launch dev server**
   ```bash
   npm run dev
   ```
